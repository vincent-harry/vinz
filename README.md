## Capstone Project

**PLANTERAID: Web-Based Planter Aid Soil Moisture Monitoring and Recommendation System with Alert Notifications Using Microcontroller Devices**<br>
*Collaborative group Capstone project*

The project combines a web dashboard with IoT sensor nodes to monitor soil moisture, temperature, relative humidity, and zone location. It presents real-time and historical readings, rule-based crop-care recommendations, and alert notifications when configured conditions are exceeded. It is a monitoring and decision-support system and does not perform automated irrigation.

- **Software stack:** C#, Blazor .NET Core 8.0, HTML, CSS, JavaScript, SQL Server, SQL Server Management Studio, Arduino IDE
- **Hardware integration:** ESP32/Lolin32, SHT31-DIS-B temperature and humidity sensor, SEN0308 capacitive soil-moisture sensor, NEO-M8N GPS module
- **My contribution:** Research and field interviews, system and database work, testing, technical documentation, and preparation of Capstone figures and progress materials
- **Portfolio:** [View the Capstone system, hardware, and testing materials](https://vhhtrayvilla-portfolio.vercel.app/#capstone)
